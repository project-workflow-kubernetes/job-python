# job-python

Dependencies to run the Makefile commands: `conda>=4.5.11, pylint>=2.1.1, docker>=18.06.1`

* Install dependencies in your machine:
```bash
make install
```

* Run project (execute `main.py`):
```bash
make run
```

* Run tests in your machine:
```bash
make test
```

* Run tests in docker:
```bash
make test-in-docker
```
